Cultivating Cultures

Using Scientific Ingenuity, Improbable Beef has undertaken the task of cultivating proteins for synthetic beef from volunteers' navels. If Improbable Beef succeeds in this endeavor, this website will allow the company to locate those volunteers whose navels featured the proteins that will need to be synthesized and manufactured.
